# 🏥 MedCore Hospital Management System

A modern desktop Hospital Management System built using **Python**, **PySide6 (Qt)**, and **SQLite** to simplify hospital administration through secure role-based access, intelligent appointment scheduling, medical record management, billing, and analytics.

---

## ✨ Features

### 👨‍⚕️ Admin Dashboard
- Doctor Management
- Patient Management
- Appointment Monitoring
- Billing Management
- Revenue Analytics
- Interactive Charts
- Hospital Statistics

### 🩺 Doctor Portal
- Secure Login
- Daily Appointment Schedule
- Patient Medical History
- Consultation Management
- Prescription Creation
- Medical Record Management

### 👤 Patient Portal
- Registration
- Secure Authentication
- Book Appointments
- Family Member Management
- Appointment History
- Medical Records
- Billing History

---

## 🔐 Security

- PBKDF2-HMAC-SHA256 Password Hashing
